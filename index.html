    crystal-gsl

    GNU Scientific Library (GSL) binding for Crystal

    Installation

    Add this to your application's shard.yml:

    dependencies:
  gsl:
    github: ruivieira/crystal-gsl

    System dependencies

    Ubuntu

    • libatlas-base-dev
    • libgsl-dev

    Fedora

    • gsl
    • gsl-devel

    Usage

    require "gsl"
